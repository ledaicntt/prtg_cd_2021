Here is a note from your PRTG Network Monitor server: you are currently using VMware sensors in your PRTG installation. Do you know all VMware sensors PRTG offers you?


With these sensors you can monitor the VMware ESX server by CPU, disk, memory and network:

  • VMware Host Hardware (WBEM) Sensor
  • VMware Host Hardware Status (SOAP) Sensor
  • VMware Host Performance (SOAP) Sensor
  • VMware Virtual Machine (SOAP) Sensor
  • VMware Datastore (SOAP) Sensor


A device template for automatic discovery of VMware ESX servers makes it easy to set up and use the VMware sensors.


If you want to run PRTG on VMware, please find a checklist in our Knowledge Base.
