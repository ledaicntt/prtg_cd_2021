Here is a note from your PRTG Network Monitor server: your PRTG Network Monitor server does not run on one of the most recent versions of Windows. Therefore you may experience PRTG slower than it can actually be.


For your best monitoring experience with PRTG we recommend Microsoft Windows Server 2012 R2 or later for PRTG Core Service and Probe Service.
For details please see our system requirements.


By moving to Windows 2012 R2 or later you can improve especially the monitoring performance via WMI up to factor of 10.
Please see our Knowledge Base for which sensor types we recommend Windows Server 2012 R2 and why.


Thanks for using PRTG!
