Dies ist ein Hinweis von Ihrem PRTG Network Monitor Server: im Augenblick nutzen Sie VMware Sensoren in Ihrer PRTG Installation. Kennen Sie schon alle VMware Sensoren, die PRTG anbietet?


Mit diesen Sensoren können Sie CPU, Festplatte, Speicher und Netzwerk Ihres VMware ESX Servers monitoren:

  • VMware Host Hardware (WBEM) Sensor
  • VMware Host Hardware Status (SOAP) Sensor
  • VMware Host Performance (SOAP) Sensor
  • VMware Virtual Machine (SOAP) Sensor
  • VMware Datastore (SOAP) Sensor


Ein Device Template für die Auto-Discovery von VMware ESX Servern macht es einfach VMware Sensoren anzulegen und diese zu nutzen.


Falls Sie PRTG selbt auf einer VMware nutzen wollen, sehen Sie sich bitte diese Checkliste in unserer Knowledge Base an.


Danke, dass Sie PRTG nutzen!
