Dies ist ein Hinweis von Ihrem PRTG Network Monitor Server: Ihr PRTG Network Monitor Server läuft im Augenlick nicht auf einer der aktuellsten Versionen von Windows. Daher ist PRTG möglicherweise langsamer als es sein könnte.


Für optimales Monitoring mit PRTG empfehlen wir Microsoft Windows Server 2012 R2 oder neuer für den PRTG Core Service und Probe Service.
Details finden Sie in unseren Systemanforderungen.


Indem Sie auf Windows 2012 R2 oder neuer migrieren, können Sie insbesondere die Monitoring Leistung via WMI um einen Faktor bis zu 10 verbessern.
In unserer Knowledge Base finden Sie Empfehlungen für welche Sensor Typen wir Windows Server 2012 R2 empfehlen und warum.
