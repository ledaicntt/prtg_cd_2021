Dies ist ein Hinweis von Ihrem PRTG Network Monitor Server: Im Augenblick nutzen Sie <#objectstatus name="totalsens" show="textraw" id="0"> Sensoren auf einem virtuellen System. Wir empfehlen dringlich von der virtuellen Maschine auf physische Hardware zu migrieren, falls möglich. Dies wird die Performance und Zuverlässigkeit Ihrer PRTG Installation deutlich verbessern.


Es gibt mehrere Gründe, warum wir Ihnen empfehlen, dass PRTG (Core-Server und Remote Probes) vor allem beim Gebrauch von <#objectstatus name="totalsens" show="textraw" id="0"> Sensoren auf einer echten Hardware laufen sollte.


Jede Sensoren Anfrage durchläuft viele Virtualisierungs-Ebenen, was Leistung in Anspruch nimmt und zudem dazu führt, dass Messungen ungenau werden. Erfahrungsgemäß funktioniert eine physische Maschine bei Gebrauch von mehr als tausend Sensoren am besten. Unsere Empfehlung zur Verwendung echter Hardware gilt sowohl für PRTG-Server als auch Remote Probes.


Falls Sie PRTG auf einer virtuellen Maschine nutzen müssen, verwenden Sie bitte maximal 2.500 Sensoren pro virtueller Maschine und nutzen Sie stattdessen mehrere PRTG-Server-Instanzen.
