Aqui está um lembrete do seu servidor PRTG Network Monitor: Atualmente você está usando <#objectstatus name="totalsens" show="textraw" id="0"> sensores em um cluster. Isso está acima de nossos limites críticos de sensores. Se você deseja monitorar mais de 2.500 sensores em um cluster de failover PRTG, nós pedimos que você entre em contato com nossa equipe de pré-venda para consulta.


Recomendamos uma única configuraçã o de failover, se você precisa de monitoramento fail-safe. Este consiste em dois servidores de núcleo PRTG, cada um trabalhando como um nó de cluster.
Em um cluster de failover PRTG, a carga de monitoramento dobra com cada nó de cluster, de modo que você encontra a metade de desempenho para cada nó de cluster adicional.
