Aqui está um lembrete do seu servidor PRTG Network Monitor: Atualmente você está usando sensores VMware na instalaçã o do seu PRTG. Você já conhece tudo que os sensores VMware PRTG oferecem-lhe?


Com estes sensores você pode monitorar o servidor VMware ESX por CPU, disco, memória e rede:

  • VMware Host Hardware (WBEM) Sensor
  • VMware Host Hardware Status (SOAP) Sensor
  • VMware Host Performance (SOAP) Sensor
  • VMware Virtual Machine (SOAP) Sensor
  • VMware Datastore (SOAP) Sensor


Um modelo de dispositivo para descoberta automática de servidores VMware ESX torna mais fácil de configurar e usar os sensores VMware.
