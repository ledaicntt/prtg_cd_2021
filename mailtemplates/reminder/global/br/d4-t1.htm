Aqui está um lembrete do seu servidor PRTG Network Monitor: O servidor PRTG Network Monitor nã o pode ser executado em uma das versões mais recentes do Windows. Portanto, poderá acontecer do PRTG ficar mais lento do que ele realmente deveria.


Para sua melhor experiência de monitoramento com o PRTG, recomendamos o Microsoft Windows Server 2012 R2 ou posterior para PRTG Core Service e Probe Service.
Para mais detalhes consulte os nossos requisitos de sistema.


Movendo-se para o Windows 2012 R2 ou posterior você pode melhorar especialmente o desempenho de monitoramento via WMI.
Consulte a nossa base de conhecimento para saber para quais sensores recomendamos o Windows Server 2012 R2 e porquê..
