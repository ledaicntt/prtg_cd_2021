Ceci est un message de votre serveur PRTG Network Monitor : vous utilisez actuellement des capteurs VMware dans votre installation PRTG. Connaissez-vous tous les capteurs VMware que PRTG propose ?


Avec ces capteurs, vous pouvez surveiller le serveur VMware ESX par CPU, le disque, la mémoire et le réseau :

  • VMware Host Hardware (WBEM) Sensor
  • VMware Host Hardware Status (SOAP) Sensor
  • VMware Host Performance (SOAP) Sensor
  • VMware Virtual Machine (SOAP) Sensor
  • VMware Datastore (SOAP) Sensor


Un modè le d’appareil pour une découverte automatique de serveurs VMware ESX rend la configuration et l’usage de ces capteurs faciles.
