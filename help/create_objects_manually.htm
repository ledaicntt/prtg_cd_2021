<%NUMBERING1%>.<%NUMBERING2%>.<%NUMBERING3%> PRTG Manual: Create Objects Manually

We recommend that you use the auto-discovery function to create a basic monitoring setup for your network. Afterwards, you can manually create devices that could not be discovered, or arrange detected devices in groups.

The procedure depends on the kind of object you want to add. Choose between:

  • Add a Group
  • Add a Device
  • Add a Sensor

Add a Remote Probe

Please see section Multiple Probes and Remote Probes for more information.
