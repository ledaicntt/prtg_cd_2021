Drag&Drop

Within the same device, drag any sensor and drop it to the place where you want to have it. The future position is highlighted. When dropping, the sensor will be moved to this position and existing sensors will be lined up after it. This is a very easy way to reposition your sensors.

Drag a sensor from one device and drop it to another to clone a sensor.

Drag and drop groups or devices to change their position or to move them into (other) groups.

Multi-Edit

You can use Multi-Edit for object settings. Hold down the Ctrl key and select multiple groups, devices, or sensors (of the same type).

In the appearing dialog, select the settings you would like to edit, change the according values and click Save. The changes will be applied to all selected objects.
